Hydra

What is Hydra?

Hydra is an experimental interpreted language which explores the intersection of several design concepts: it has a prototype based object-system inspired by the self language, but with a multiple dispatch system inspired by CLOS. When combined with lispy syntax and macros, this leads to interesting metaprogramming and self-modification opportunities.

Learn

I have written a small wiki on github, which is generally up-to date, although there are occasional discrepencies between how something is described vs how it actually behaves.

Get it

Windows

Hydra has no installer, but you can grab the raw binary here or follow the build instructions on the github page. You'll need conan, CMake and the Microsof's visual C++ compiler to build.

Linux

As with windows, there is no distribution option via package managers or similar, but you can buid from source, following the instructions on the github page. You'll need CMake to build it.

OSX

Hydra has not formally been compiled/tested on OSX, but if you are familiar with C++ compilation on OSX, you might wish to try adapting the build instructions on the github page.
