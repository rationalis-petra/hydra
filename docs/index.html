Hydra

What is Hydra?

Hydra is an experimental interpreted language made by one person (me), which explores the intersection of several design ideas: it has an object-system inspired by the self language, but with a multiple dispatch system borrowed from CLOS. This is combined with lispy syntax and macros, and this should hopefully lead to interesting metaprogramming and self-modification opportunities.

Learn

I have written a small wiki on github, which is generally up-to date, although there are occasional discrepencies between how something is described vs how it actually behaves.

Get it

Windows

Hydra has no installer, but you can either grab the raw binary here or follow the build instructions on the github page. You'll need conan, CMake and the Microsof's visual C++ compiler to build.

Linux

As with windows, there is no distribution option via package managers or similar, but you can buid from source, following the instructions on the github page. You'll need CMake to build it.

OSX

Unfortunately, I have not formally compiled/tested hydra on OSX, so I am not aware of whether or not it works, but you can try adapting the build instructions on the github page if you are familiar with c++ compilers/conan on OSX.
